# YaCy Search Server
A Compiled version of YaCy v1.925/9749 for Windows and Linux.

Source see https://github.com/yacy/yacy_search_server

Download

http://165.22.240.144/YaCy_1.925_9749.zip


Windows users

Unzip to c:\

Double click  Install yacy 1.925_9749 icon.


Linux users

sudo apt install openjdk-8-jdk

cd YaCy

./startYACY.sh

Will open browser to http://localhost:8090
